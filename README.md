**Finance Dashboard**

A modern, responsive Finance Dashboard built using React and Redux Toolkit. This project
demonstrates clean UI design, structured state management, and interactive data visualization to
help users track and understand their financial activity.

**Live Demo:** _https://zorvyn-finance-dashboard.web.app/_

**Overview**
  - View overall financial summary
  - Analyze spending trends and patterns
  - Manage transactions efficiently
  - Switch between roles (Viewer/Admin)
  - Gain meaningful financial insights

**Features**
**Dashboard Overview**
  - Summary cards: Total Balance, Income, Expenses
  - Trend visualizations (sparklines)
  - Category-wise spending breakdown

**Transactions Management**
  - Transaction list with Date, Amount, Category, Type
  - Search by category or description
  - Filter by month and year
  - Sorting options

**Role-Based UI**
  - Viewer: Read-only access
  - Admin: Add, edit, delete transactions
  - Role toggle available in UI

**Insights Section**
  - Highest spending category
  - Monthly comparison
  - Smart observations

**UI / UX Highlights**
  - Clean and modern interface
  - Responsive design
  - Dark and light mode
  - Smooth animations
  - Handles empty states gracefully

**Tech Stack**
_Category Tools_
  - Frontend React 18 (Vite)
  - State Management Redux Toolkit
  - Charts ApexCharts
  -  UI Material UI + CSS
  - Icons React Icons
  - Backend json-server

**Setup Instructions**
  - Install dependencies: npm install
  - Start backend: npm run server
  - Start frontend: npm run dev

**Additional Features**
  - Export transactions (CSV/JSON)
  - Theme persistence using localStorage
